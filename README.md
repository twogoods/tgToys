# go玩具
用go实现的几个有意思的小玩意~

## 快递查询
terminal下的快递查询服务。 
![快递查询](http://oepm97cib.bkt.clouddn.com/express.png)
## 天气查询
terminal下的天气查询服务，可以根据ip自动定位城市，也可以手动输入城市。
![天气查询](http://oepm97cib.bkt.clouddn.com/weather.png)

## 安装 & 使用
安装:`go get github.com/twogoods/golib/gohttp`  
使用： `TgToys express <expressNum>` `TgToys.go weather -<city>`
